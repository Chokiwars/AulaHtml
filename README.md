#  Aula HTML

Este repositório contém exemplos e exercícios práticos para aprendizado de HTML.

---

##  Como executar o projeto

Este é um projeto estático, então você pode rodar de forma simples:

###  Opção 1: Abrir direto no navegador

* Localize o arquivo `index.html`
* Clique duas vezes ou abra com seu navegador

###  Opção 2: Usando Live Server (recomendado)

1. Instale a extensão **Live Server** no VS Code
2. Clique com o botão direito no arquivo `index.html`
3. Selecione **"Open with Live Server"**

---

##  Estrutura do projeto

* `index.html` → Página principal
* Outros arquivos `.html` → exemplos e exercícios das aulas

---

##  Pré-requisitos

* Navegador atualizado (Chrome, Edge, etc.)
* (Opcional) VS Code + extensão Live Server

---

##  Objetivo

Este projeto tem como objetivo auxiliar no aprendizado de HTML por meio de exemplos simples e práticos.
